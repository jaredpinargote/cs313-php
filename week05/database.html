MGC: Main Game Client, The client that creates the game, displays instructions, provides countdown timer, displays drawings/captions/combos, and displays winner. Ideally would be hooked up to a TV for a game.

PC: Player Client, The client that a player would use to join a game, submit drawings/captions/combos and vote on the best ones

Create New Game

MGC: Creates a new game by sending a GET request. Gets the newly created game ID and room code, saves game ID as cookie, displays room code

Create New Player And Join Game

PC: Player enters room code and user name, submits, and recieves confirmation they are in. Cookie is set with playerID. Server sends eventAlert to MGC to display new player

View Players in game
